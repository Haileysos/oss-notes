1. commit 상태 출력　　　　　　　　　　git log
2. c1으로 revert　　　　　　　　　　　　git revert c1.hash
3. 파일 상태 출력　　　　　　　　　　　　git status
4. 파일제거 명령 (파일 제거로 충돌 해결)　　git rm &lt;fileName&gt;
5. revert 계속 명령　　　　　　　　　　　git revert --continue
6. commit 상태 출력　　　　　　　　　　git log
