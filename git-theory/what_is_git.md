git이란
- 컴퓨터 파일의 변경사항을 추적하고, 여러명의 사용자들 간에 해당 파일들의 작업을 조율할 수 있도록 도와주는 소프트웨어
- version control system
주요기능
- 버전관리 : 코드, 문서 등 파일의 변경이력 추적가능, 각 변경사항에 대해 고유한 커밋 아이디생성해 기록 하고 과거버전으로 쉽게 되돌리기 및 특정시점학인가능 -백업:분산형구조로인해 로컬저장소에 전체데이터가 저장됨.중앙서버가손상되어도로컬저장소를통해ㅈ복구가능 -협업:병렬로작업할수잇도록브랜치생성및관리지원

SSH(secure shell)
☆주로 원격서버에 접속하여 작업을 수행하거나 파일을 전송할때 사용
컴퓨터간안전하게통신할수잇도록하는네트워크프로토콜(원격서버)
통신내용은암호화되어전송되기때문에 해킹위험적음
HTTP/HTTPS
사용자가 웹브라우저에 유알엘을입력하면 HTTP/HTTPS 프로토콜통해 웹서버와 통신
HTTP는 데이터를 암호화하지않음
HTTPS는 SSL을 사용해 데이털 암호화하여 연결제공
